# v1.4.5 — Reconnect fixes

## What's new?

- Rejoining a room (or minimising and coming back) no longer puts you out of sync — chat history, queue, and room settings all restore properly from the server
- Chat now shows a "Reconnecting…" message when the connection drops instead of silently eating your messages
- Fixed chat and room settings stopping after a rejoin
- Discord users should no longer randomly end up in different rooms after a disconnect
- Old chat messages now roll off properly — only the last 200 are kept, same as the server
