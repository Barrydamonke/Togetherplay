# v1.0 — Initial Release

The first release of Togetherplay.

## Rooms & Sync

- Create a room and receive a 4-digit PIN to share with friends
- Real-time playback sync across all members (within 2 seconds)
- Host controls playback — guests follow along automatically
- Automatic host reassignment when the host leaves, with a toast notification
- 60-second grace period — rooms survive briefly after everyone disconnects
- Automatic reconnection after a dropped or interrupted connection

## Join Flow

- Browse active rooms directly from the join screen
- See who is already watching in each room
- Empty rooms can be joined without entering a PIN
- Manual PIN entry still available for rooms not shown in the list

## Video Player

- HLS streaming via Jellyfin with direct-stream fallback
- Full custom controls: seek bar, volume, mute, fullscreen
- Subtitle track selection with live rendering
- Buffering indicator and playback error messages

## Queue

- Browse and search your entire Jellyfin library
- Add, remove, and drag to reorder queued items (host only)
- Automatically advances to the next item when one ends

## Chat

- Live per-room chat
- Last 200 messages kept per session

## Other

- Dark and light theme, persisted across visits
- Username remembered across visits
- Admin panel to configure your Jellyfin server connection
- Animated Jellyfin poster background on the landing page
- Extra posters toggle (12 or up to 100)

## Coming in a future release

- Mobile-responsive layout
- Touch-friendly queue reorder
- Protection against bad actors (rate limiting, input validation)
- Persistent room storage so a server restart doesn't kill active sessions
- Upload service — there's a config field for it in the admin panel but no UI to use it.
- Rate limiting / spam protection — no throttling on socket events or API endpoints.
