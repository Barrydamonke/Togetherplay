# v1.4.2 — Sidebar update

## What's new?

- The sidebar now has collapse arrows on each section (Watching, Queue, Chat) — click the header to toggle them. When a section is collapsed, the others fill the freed space.
- The sidebar is resizable — drag the left edge to make it wider or narrower.
- The settings modal now scrolls properly on smaller screens instead of going off the page.
- Discord fix: the fullscreen button is replaced with a sidebar toggle when running inside a Discord activity, since native fullscreen is blocked in the iframe.
- Various Discord streaming fixes.
